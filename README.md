# Peta Sumber Gempa

Peta Interaktif Sumber Gempa (Sesar dan Subduksi) Indonesia.

## Live Demo
Akses peta interaktif di sini: [https://ftharyanto.github.io/Peta-Sumber-Gempa/](https://ftharyanto.github.io/Peta-Sumber-Gempa/)

## Deskripsi
Repositori ini berisi versi statis dari Peta Sumber Gempa yang dapat dijalankan langsung melalui GitHub Pages. Data yang digunakan bersumber dari file JSON di folder `data/`.

## Fitur
- Visualisasi Sesar dan Subduksi.
- Basemap Satelit dan Layer Control.
- Informasi detail pada setiap fitur melalui popup.
